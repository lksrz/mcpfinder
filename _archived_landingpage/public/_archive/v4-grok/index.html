For Developers

REST API

Integrate with our simple and secure API to register and search for MCP tools programmatically.

                        POST /api/v1/register
GET /api/v1/tools/:id
GET /api/v1/search?tag=&q=

CLI Tool

Use our command-line interface for quick and easy tool registration.

                        mcp-cli register path/to/mcp.json
