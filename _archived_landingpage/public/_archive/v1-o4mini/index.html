Core Features

Tool Registration

Authenticate and register tools with HMAC signature and schema validation.

Tool Discovery

Search and filter registered tools via REST endpoints by tag or keyword.

Health Monitoring

Automated uptime checks and status tracking for all registered tools.

API Endpoints

POST /api/v1/register

Register a tool using a signed HMAC manifest. Returns success and id.

GET /api/v1/tools/:id

Fetch the full manifest and status for a specific tool.

GET /api/v1/search?tag=&q=

Search tools by tag or keyword. Returns a list of summaries.

Command Line Interface

Use the mcp-cli to register your tool manifests:

mcp-cli register path/to/mcp.json

The CLI computes the HMAC signature and sends it to the registry.
