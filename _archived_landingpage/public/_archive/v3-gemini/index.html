MCP Registry

Discover, Register, and Integrate AI Tools Seamlessly.

A lightweight, serverless hub for the Model Context Protocol (MCP) ecosystem, making it easy to find and share capabilities between AI agents and tools.

Get Started Read Docs

Connecting the AI Ecosystem

As AI agents and specialized tools proliferate, discovering and connecting them becomes complex. MCP Registry provides a central, standardized directory, simplifying integration and fostering collaboration within the Model Context Protocol landscape.

Core Features

Tool Registration

Easily register your MCP-compliant tools via a simple CLI command or REST API. Define capabilities, authentication, and metadata using the standard `mcp.json` manifest.

Tool Discovery

Search for tools using tags or keywords through a straightforward REST API. Find the capabilities you need for your AI agents quickly and efficiently.

Health Checks

Automatic health monitoring ensures the registry lists active and responsive tools. Status indicators provide reliability information at a glance.

Get Started

Registering your tool is simple with the MCP CLI.

# 1. Install the CLI (Example command)
npm install -g @mcpfinder/cli

# 2. Create your mcp.json manifest
{
  "name": "MyExampleTool",
  "description": "Does something awesome.",
  "url": "https://my-tool.example.com/mcp",
  ...
}

# 3. Register your tool
mcp-cli register ./path/to/your/mcp.json

Refer to the full documentation for API details and advanced usage.
