API At A Glance

POST /api/v1/register

Submit an mcp.json manifest with HMAC auth. Returns your new toolId.

GET /api/v1/tools/:id

Fetch a full manifest including live status.

GET /api/v1/search?tag=&q=

Search tools by tag or keyword and receive concise summaries for quick filtering.

Zero-friction CLI

Ship new capabilities in seconds using mcp-cli.

# Install (example)
npm install -g @mcpfinder/cli

# Create your manifest (mcp.json)
{
  "name": "MyExampleTool",
  "description": "Does something awesome.",
  "url": "https://my-tool.example.com/mcp",
  "protocol_version": "MCP/1.0",
  "capabilities": [ { "name": "doThing", "type": "tool" } ],
  "tags": ["demo"],
  "auth": { "type": "api-key" }
}

# Register
mcp-cli register ./mcp.json
