The Central Hub for AI Tool Discovery

MCP Registry connects AI agents with the tools they need. Publish your MCP-compatible tools or discover capabilities for your AI agents in our lightweight, serverless registry.

Register Your Tool Explore Registry
mcp-cli.js
const mcpCli = require('mcp-cli');
// Register your MCP tool with just one command
mcpCli.register('./mcp.json')
  .then(response => {
    console.log(`Tool registered! ID: ${response.id}`);
  });

Why Use MCP Registry?

A lightweight serverless platform connecting AI agents with the tools they need

Tool Registration

Easily register your MCP-compatible tools via CLI or REST API, making them discoverable by AI agents worldwide.

Tool Discovery

Find the perfect tools for your AI agent using our powerful search API with tagging and categorization.

Health Monitoring

Continuous health checks ensure your agents only connect to available and functioning MCP tools.

Lightweight & Serverless

Built on Cloudflare Workers with KV and R2 storage for maximum performance, reliability and minimal overhead.

Secure

HMAC authentication for registration and simple rate limiting to protect the ecosystem.

Agent Friendly

Designed for AI agents to autonomously discover, evaluate, and integrate new capabilities.

How It Works

Simple workflow for both tool publishers and AI agents

For Tool Publishers

  • 1

    Create Your MCP Tool

    Build a server implementing the MCP protocol to provide capabilities to AI agents.

  • 2

    Define Your Manifest

    Create an mcp.json file describing your tool's capabilities, authentication requirements, and other metadata.

  • 3

    Register with MCP Registry

    Use our CLI or REST API to register your tool, making it discoverable by AI agents.

For AI Agents

  • 1

    Search for Capabilities

    Query the registry's search API to find tools that match your requirements.

  • 2

    Get Tool Details

    Retrieve complete tool metadata including endpoint URL, capabilities, and authentication requirements.

  • 3

    Connect & Utilize

    Connect directly to the tool using the MCP protocol and expand your capabilities.

Simple & Powerful API

Easy-to-use endpoints for registration, discovery, and search

API Examples
# Register a new tool
POST /api/v1/register
Headers: Authorization: HMAC <signature>
{
  "name": "WeatherMCP",
  "description": "Get current weather and forecasts",
  "url": "https://weather-mcp.example.com",
  "protocol_version": "MCP/1.0",
  "capabilities": [
    { "name": "getCurrentWeather", "type": "resource" },
    { "name": "getForecast", "type": "resource" }
  ],
  "tags": ["weather", "forecast"]
}
# Search for tools
GET /api/v1/search?tag=weather&q=forecast
# Get tool details
GET /api/v1/tools/:id
