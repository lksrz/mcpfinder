For AI Assistants

Self-Expanding AI Assistants

MCP Finder Server acts as an "App Store for Agents" - enabling AI assistants to discover and install new tools on demand based on user requests.

As an AI assistant with access to MCP Finder Server, you can:

  • Recognize when you lack capabilities needed for a user request
  • Search for tools that provide those capabilities
  • Evaluate the best tool for the job based on descriptions and capabilities
  • Install the tool directly into your client's configuration
  • Begin using the new capabilities immediately

This enables a seamless experience where users don't need to know about tool installation or configuration - they simply ask for what they need, and you adapt to provide it.

The MCP Finder Server tools are designed to be used by AI assistants in Claude, Cursor, VS Code, and other MCP-compatible clients.

MCP Tools
# 1. Find servers matching a query and/or tag
search_mcp_servers({
  "query": "weather forecast",
  "tag": "weather"
})
Returns array of matching MCP servers with name, description, URL, status
# 2. Get complete details for a specific server
get_mcp_server_details({
  "id": "weather-api-12345"
})
Returns full manifest with capabilities, authentication, and setup hints
# 3. Install a server into client configuration
add_mcp_server_config({
  "server_id": "weather-api-12345",
  "client_type": "cursor"
})
Adds server to client config file (or config_file_path for custom location)
# 4. Remove a server from client configuration
remove_mcp_server_config({
  "server_id": "weather-api-12345",
  "client_type": "cursor"
})
Removes server from client config file
